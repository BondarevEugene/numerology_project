# Human State

Human State — центральная сущность платформы Genesis.

В отличие от HR систем, Genesis хранит не профессию.

Genesis хранит состояние человека.

Любое событие:

- книга
- привычка
- спорт
- обучение
- стресс
- работа
- отдых

изменяет Human State.

Именно он используется:

- Prediction Engine
- Career Engine
- Recommendation Engine
- AI Advisor
- Digital Twin
