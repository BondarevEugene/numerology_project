✨Твой Путеводитель в Мир Кода✨

Привет! Давай создадим твой чудесный сервис вместе. Это будет легко и очень мило! 🌸

🧶Шаг 1: Подготовка гнездышка

Создай папку для проекта. Тебе понадобятся "ингредиенты" (библиотеки). Открой терминал и шепни ему:

pip install flask flask-sqlalchemy fpdf2 pandas openpyxl

И не забудь положить в папку файл DejaVuSans.ttf, чтобы котик-принтер понимал русский язык!

📄Шаг 2: Сердечко программы (app.py)

Это главный файл. Он хранит данные в базе clients.db и делает магические расчеты.

# Кусочек кода для базы данных
class Client(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    name = db.Column(db.String(100))
    email = db.Column(db.String(100))
    birth_date = db.Column(db.String(20))

🎀Шаг 3: Красивое личико (index.html)

Положи этот файл в папку templates. Мы используем три окошка для даты рождения, как ты и просила!

Важно для красоты

Мы используем розовые акценты и закругленные края, чтобы пользователям было уютно.

✉️Шаг 4: Почтовый голубь

Для бесплатной отправки используй функцию smtplib.

Няшный совет: В Gmail создай специальный "Пароль приложения", чтобы всё работало как по маслу!

🚀Шаг 5: Полет на облачко (Удаленка)

Когда наиграешься локально, мы перенесем проект на Ubuntu. Там мы позовем на помощь:

  • Nginx — наш швейцар на входе.
  • Gunicorn — неутомимый моторчик под капотом.
